## Endpoint Description

This endpoint allows you to check an agent's system message.

### Query Parameters

- `uid` (required, string): The unique identifier for the user.

### Example Request

`GET https://api.zerobot.ai/v1/chat/system_message?uid={uid}`

Users need to replace `{uid}` with the actual `uid` value before sending the request. Make sure the Mintlify configuration prompts for this parameter in the API test call interface.
