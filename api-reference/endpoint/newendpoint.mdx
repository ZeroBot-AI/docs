## Endpoint Description

This endpoint allows you to create a new user account by sending a POST request to the `/users` endpoint. The request body should include the necessary user information, such as name, email, and password.

### Request Parameters

<RequestExample>
  ```yaml name: John Doe email: john@example.com password: secretpassword```
</RequestExample>
Response
<ResponseExample>
```json
{
  "id": "123",
  "name": "John Doe",
  "email": "john@example.com"
}
```
</ResponseExample>
Error Responses
<ResponseExample status={400}>
```json
{
  "error": "Invalid request payload"
}
```
</ResponseExample>
<ResponseExample status={409}>
```json
{
  "error": "Email address already exists"
}
```
</ResponseExample>
